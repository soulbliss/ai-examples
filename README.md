## Getting Started

1. Install the pakages

```
pnpm i
```

2. Create env file

```
touch .env.local
```

3. Add key in `.env.local` file

```
OPENAI_API_KEY=xxxxxxxxx
```

4. Run the server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
